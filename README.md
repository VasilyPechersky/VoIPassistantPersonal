VoIP call assistant is a Desktop APP, that Holds local Phone book and allows to call from IP phones, which allowing ActiveURI http control.

Phone vendors : Yealink, Fanvil.

GUI stack is Qt5. Programming language is Python 3.

<b>Setup instructions:</b>

Requirements for current version: Phone must use http (not https)

Both Computer and IP Phone must have static IP.

<b>IP Phone setup :</b>

Fanvil: 

Phone Settings -> action -> incomming call-> http://"your computer IP":18025/call?state=ring&in=$remote_number

Phone Settings -> Features -> Restrict Active URI Source IP: "your computer IP"

Yealink:

See instructions at https://support.yealink.com/document-detail/f178164a87b14603a557ef6386c78462

Features -> Action URL -> incomming call-> http://"your computer IP":18025/call?state=ring&in=$remote_number

<b>Program setup :</b>

As for version from 05/07/2026

In Settings: provide phone IP, Phone Username, Password. Fields Extension, Token are still not in use

Action buttons : Provide name and System path to executeble (or just name if in system path) to run it from program. If your program supports arguments $N provides caller Number

To import contacts, Gcontactpuller (py or executable) must be in the same directory. Tested by importing Contacts from VCF file, generated by Google Contacts.
